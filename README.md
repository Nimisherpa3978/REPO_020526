
# ✋ Virtual Hand Gesture Keyboard

Control your keyboard using hand gestures via webcam.

## 🚀 Features
- Detects hand using MediaPipe
- Counts fingers
- Maps gestures to keyboard input
- Real-time interaction

## 🧠 Tech Stack
- Python
- OpenCV
- MediaPipe
- PyAutoGUI

## 🎮 Controls
| Fingers | Action |
|--------|--------|
| 1 | A |
| 2 | B |
| 3 | C |
| 4 | SPACE |
| 5 | ENTER |

## ▶️ Run
```bash
pip install -r requirements.txt
python main.py
